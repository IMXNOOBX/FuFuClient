## 🪁 Simple Use

***IntelliJ***

> gradlew genIdeaWorkspace

***Eclipse***

> gradlew genSources eclipse

***Other IDE's***

Use [this link](https://fabricmc.net/wiki/tutorial:setup) for more information.

***Compile***

> gradlew build
